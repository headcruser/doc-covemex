# Plataforma web Covemex

## Instalación de la plataforma
A continuación se explicará de manera breve, la configuración básica que debe de tener el proyecto para poder ejecutar el proyecto.


> Se debe tener instalado en el servidor el driver [Microsoft ODBC ](https://docs.microsoft.com/en-us/sql/connect/odbc/linux-mac/installing-the-microsoft-odbc-driver-for-sql-server?view=sql-server-2017
) para php

## Requerimientos
Para tener un entorno de desarrollo adecuado, debe tener instalado los siguientes paquetes

* Apache 2
* PHP >= 7.1.3
* PDO PHP Extension
* Composer
* NodeJS
* Git
